About This Project

What is Lyric Explorer?

Lyric Explorer is a small interactive web app designed to help music lovers explore, analyze, and discover new lyrics. It was created as a portfolio project to demonstrate modern web development skills using HTML, CSS, and JavaScript.

Key Features

  • 📖 View curated lyrics
  • 🔍 Keyword search and highlight
  • 📊 Real-time word and character stats
  • 🎥 Watch official music videos

Built With

  • 🛠 HTML, CSS, JavaScript
  • 🎨 Custom UI with modern effects
  • 📱 Responsive design
